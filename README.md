# 🌦 Weather Web Application

A simple and interactive weather web application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using a public API.

---

## Features

* Search weather by city name
* Displays temperature in Celsius
* Shows weather condition (e.g., Clear, Rainy)
* Displays humidity
* Dynamic background based on weather
* Weather icons from API
* Error handling for invalid cities

---

## 🛠 Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Logic and API handling
* **OpenWeatherMap API** – Fetch real-time weather data

---


---

## How to Run the Project

1. Clone or download this repository
2. Open the project folder
3. Open `index.html` in your browser

---

## API Setup

1. Sign up on OpenWeatherMap
2. Get your free API key
3. Replace the API key in `script.js`:

```javascript
let apiKey = "YOUR_API_KEY";
```
